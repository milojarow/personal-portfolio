# rolandoahuja.com

Personal portfolio + knowledge garden. Bilingual EN/ES.

## Development

```bash
npm install
npm run dev
```

Opens at `http://localhost:3000/` (English) and `http://localhost:3000/es` (Spanish).

## Stack

Next.js 16 · React 19 · Tailwind 4 · DaisyUI 5 · next-intl · gray-matter + remark

## Deploy

Vercel-hosted. `vercel --prod` or push to main.
